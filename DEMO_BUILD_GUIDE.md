# 📱 دليل بناء النسخة التجريبية - HUS App Demo

## 🎯 النسخة التجريبية جاهزة للبناء!

تم إنشاء نسخة تجريبية كاملة من التطبيق تعمل ببيانات وهمية **بدون الحاجة لإعدادات Firebase أو ZegoCloud**.

## ✅ ما تم تحضيره

### 🔧 **الخدمات التجريبية**
- `MockAuthService` - مصادقة وهمية مع مستخدمين تجريبيين
- `MockRoomService` - غرف صوتية وهمية مع بيانات حقيقية
- `MockMediaService` - تشغيل وسائط وهمي (YouTube + ملفات صوتية)

### 📱 **الواجهات المحدثة**
- `main.dart` - محدث للعمل مع الخدمات التجريبية
- `DemoLoginScreen` - شاشة دخول تجريبية أنيقة
- جميع الشاشات تعمل مع البيانات الوهمية

### 🎵 **البيانات التجريبية**
- **5 مستخدمين وهميين** مع صور وبيانات كاملة
- **4 غرف صوتية متنوعة** (موسيقى، أصدقاء، تعليم، ألعاب)
- **مقاعد تفاعلية** مع مستخدمين وهميين
- **رسائل دردشة** في كل غرفة
- **محتوى وسائط وهمي** (YouTube + ملفات صوتية)

## 🚀 خطوات البناء السريع

### 1. **تحضير المشروع**
```bash
# الانتقال لمجلد المشروع
cd hus_project/frontend

# نسخ إعدادات النسخة التجريبية
cp pubspec_demo.yaml pubspec.yaml

# تثبيت التبعيات
flutter pub get
```

### 2. **بناء APK تجريبي**
```bash
# تنظيف المشروع
flutter clean
flutter pub get

# بناء APK تجريبي
flutter build apk --debug --target=lib/main.dart

# أو للإصدار النهائي
flutter build apk --release --target=lib/main.dart
```

### 3. **تثبيت واختبار**
```bash
# تثبيت على الجهاز المتصل
flutter install

# أو تثبيت يدوي
adb install build/app/outputs/flutter-apk/app-debug.apk
```

## 📱 تجربة المستخدم

### 🚀 **عند فتح التطبيق**
1. شاشة تحميل أنيقة (2 ثانية)
2. شاشة دخول تجريبية مع معلومات
3. اختيار "دخول تجريبي" أو "Google تجريبي"
4. دخول فوري للشاشة الرئيسية

### 🏠 **في الشاشة الرئيسية**
1. عرض 4 غرف متنوعة
2. معلومات كل غرفة (عدد المستخدمين، النوع)
3. إمكانية الدخول لأي غرفة فوراً

### 🎵 **داخل الغرفة**
1. مالك الغرفة في الأعلى مع التاج
2. 8 مقاعد مرتبة (4×2) مع مستخدمين وهميين
3. دردشة حية مع رسائل موجودة
4. أزرار تحكم كاملة (إعدادات، وسائط، مغادرة)
5. إمكانية طلب مقعد والانضمام للمحادثة

### 🎬 **تشغيل الوسائط**
1. إضافة فيديو YouTube (محاكاة)
2. رفع ملف صوتي (محاكاة)
3. مشغل متقدم مع أدوات تحكم
4. قائمة انتظار قابلة للإدارة
